🍽️ Restaurant Website - Innovexa Internship Project 1

🚀 Project Title
Responsive Restaurant Website

📌 Overview
This project is developed as part of the **Innovexa Catalyst Full Stack Development Internship - Project 1**.  
It is a responsive and user-friendly restaurant website built using **HTML, CSS, and JavaScript**.

The main goal of this project is to create a modern, clean, and interactive website that works smoothly across all devices.

🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript

🎯 Project Objective
- Build a responsive website that works on all devices (mobile, tablet, desktop)
- Create a clean and modern user interface
- Improve navigation and user experience
- Use structured and maintainable code

✨ Features
- Fully responsive design 📱💻
- Clean and modern UI 🎨
- Interactive elements using JavaScript ⚡
- Fast loading performance 🚀
- Cross-browser compatibility 🌐
- Easy to maintain code structure 🛠️
